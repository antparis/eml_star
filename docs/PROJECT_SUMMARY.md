## 1. THE MATHEMATICAL CORE

### 1.1 Starting point: Odrzywołek (2026)

arXiv:2603.21852v2 — "All elementary functions from a single operator"

The operator `eml(x,y) = exp(x) − ln(y)` with constant 1 generates ALL classical
elementary functions. Grammar: `S → 1 | eml(S,S)`. This is the continuous analogue
of the NAND gate in Boolean logic.

### 1.2 The structural limitation we identified

eml is holomorphic by construction. Therefore:

- **Theorem 2.1 (Holomorphic closure):** Every element of E = {eml,1} is holomorphic.
- **Corollary 2.2 (UNCONDITIONAL):** Re, Im, conj ∉ E. Impossible by any finite eml tree.

This means eml alone cannot compute complex conjugate z̄, Re(z), Im(z), or |z|².
This is not a limitation of depth — it is a structural impossibility.

### 1.3 Our contribution: eml★

```
eml★(x, y) = exp(x) − ln(conj(y))
```

A single conjugate on the second argument. Results:

| Result | Depth | Status |
|---|---|---|
| Cor. 2.2: Re, Im, conj ∉ {eml,1} | — | UNCONDITIONAL |
| Th. 3.1: z̄ = 1 − eml★(0, eml(z,1)) | rel. 2 | CONDITIONAL: Im(z) ∈ [−π, π) |
| Th. 3.2: exact domain [−π, π) | — | CONDITIONAL, boundary proved via SymPy |
| Th. 4.3: {eml, eml★, 1} dense in C(K, ℂ) | — | Proved via Stone–Weierstrass, conditional on branch-safety lemma |

### 1.4 Theorem 4.3 — Exact wording

**Theorem 4.3.** *Let K ⊂ {z ∈ ℂ : Im(z) ∈ [−π, π)} be a compact set on which the
arithmetic trees realizing +, ×, and conj via eml/eml★ are branch-safe (i.e., all
intermediate arguments of ln remain in ℂ \ {0} along the principal branch). Then
the algebra generated by {eml, eml★, 1} is dense in C(K, ℂ).*

**Remark.** Branch-safety of the arithmetic trees K=13 (addition) and K=20
(multiplication) inherited from Odrzywołek's compiler is verified numerically
at 60 decimal places on grids and 1000 random points inside the strip
(cf. `branch_safety_final.py`, Zenodo DOI:10.5281/zenodo.19183008). A formal proof
via interval analysis on Im(·) of every intermediate node remains open.

## 5. WHAT IS RIGOROUSLY ESTABLISHED

**Unconditional (no caveats):**
- Theorem 2.1 — Every finite eml-composition is holomorphic
- Corollary 2.2 — Re, Im, conj cannot be expressed by eml alone

**Conditional on Im(z) ∈ [−π, π):**
- Theorem 3.1 — z̄ = 1 − eml★(0, eml(z,1)) at depth 2
- Theorem 3.2 — exact domain limitation
- Theorem 4.3 — Stone–Weierstrass density (also conditional on branch-safety)

**Numerically verified at 60 decimal places:**
- Branch-safety of K=13 (+) and K=20 (×) trees
- Conjugate formula on 10,000 random points in the strip
- Folding property on 10,000 out-of-strip points

## 6. CAVEAT — NORMALIZED PHYSICAL STATES

The eml★ formulas for |ψ|² apply to wavefunctions bounded in sup-norm by
e^π ≈ 23 (e.g., normalized plane waves on bounded domains, coherent states
with bounded amplitude). General L²(ℝⁿ) states with arbitrary peaks fall
outside the strip and require domain decomposition (left to future work).

## 7. ZENODO ARCHIVE

DOI: 10.5281/zenodo.19183008  
File: `eml_compiler_v4.py`

```python
def eml_zero():    return eml(ONE, eml(eml(ONE, ONE), ONE))                    # K=3
def eml_neg(z):    return eml(eml(ONE,eml(eml(ONE,eml_zero()),ONE)),eml(z,ONE)) # K=8
def eml_sub(a,b):  return eml(eml_ln(a), eml_exp(b))                           # K=5
def eml_add(a,b):  return eml_sub(a, eml_neg(b))                               # K=13
def eml_mul(a,b):  return eml_exp(eml_add(eml_ln(a), eml_ln(b)))               # K=20
```

These are compiler-style constructions (Odrzywołek's Section 4.1).
